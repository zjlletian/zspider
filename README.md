ZSpider分布式网站爬虫  开发日志 verison 1.3.0

--------------------- 运行前提 ---------------------

1.Elasticsearch-php基于composer构建，运行composer install --no-dev 安装

2.使用pecl install mongo安装mongodb扩展，并添加extension=mongo.so到php.ini

完成后台管理web界面。

完成关键词搜索web api。

将队列独立出来形成服务。


--------------------- 优化任务 ---------------------

优化es的文本分析器。

完善特殊网站的规则。
