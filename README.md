ZSpider分布式网站爬虫  开发日志 verison 7.5.0

--------------------- 运行前提 ---------------------

1.运行脚本需要screen与awk工具支持。

2.php版本>=5.4。

3.使用Elasticsearch-php作为ES连接工具。https://github.com/elastic/elasticsearch-php


--------------------- 开发任务 ----------------------

完成关键词搜索web api。

任务日志图形化统计。


--------------------- 优化任务 ---------------------

优化es的文本分析器。

完善特殊网站的规则，最好使用正则表达式操作。
